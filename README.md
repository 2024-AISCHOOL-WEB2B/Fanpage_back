# GoodSwap---Back

"K-pop 팬들이 다른 팬들과 즐겁게 소통하고, 독특한 팬메이드 상품을 발견하며 안전한 환경에서 구매할 수 있는 커뮤니티 기반 플랫폼입니다."

## 개발 환경
- Java Version: 17
- IDE: IntelliJ IDEA
- Framework: Spring Boot 3.2.10

## 기술 스택
- Server: Apache
- Database: MySQL
- WAS: Tomcat
- ORM: JPA

## 주요 기능
- Redis를 활용한 JWT 토큰 관리로 인증 및 세션 처리 성능 최적화
- AWS S3를 이용한 이미지 파일 저장 및 관리
- 비동기 통신을 이용한 카드 정보 및 결제 정보 처리
